# OpenRaft-SurrealKV

**基于 OpenRaft + SurrealKV 的高性能分布式一致性 KV 存储系统**

采用 **SurrealKV 0.20** 作为统一存储后端（日志 + 状态机 + 快照），结合 **OpenRaft 0.10** 实现强一致性 Raft 共识。

---

### ✨ 核心特性

- **统一存储**：SurrealKV 同时承担 Raft Log、StateMachine、Snapshot
- **原生 Checkpoint**：零拷贝硬链接快照，构建 O(1)，恢复秒级
- **增量 + Hybrid Delta 快照**：后台异步合并策略，传输量降低 80%+
- **Tonic gRPC 流式传输**：二进制 + 64KB chunk，支持大快照
- **HTTP API**：`/write`、`/read`、`/membership`、`/change_membership`
- **Prometheus 指标**：latency histogram、error counter、SurrealKV 内部指标
- **Graceful Shutdown**：支持 Ctrl+C / SIGTERM 优雅退出
- **Docker & docker-compose**：一键三节点集群部署

### 快速开始

#### 1. 本地三节点启动（推荐）

```bash
git clone https://github.com/yourname/openraft-surrealkv.git
cd openraft-surrealkv
chmod +x examples/3nodes-local.sh
./examples/3nodes-local.sh
```

#### 2. Docker 单节点

```bash
docker build -f docker/Dockerfile.multi -t openraft-surrealkv .
docker run -p 50051:50051 -p 8080:8080 -v $(pwd)/data:/data openraft-surrealkv
```

#### 3. Docker 三节点集群

```bash
docker compose -f docker/docker-compose.yml up --build
```

### HTTP API

| 方法   | 路径                   | 功能            | 示例                                              |
|------|----------------------|---------------|-------------------------------------------------|
| POST | `/write`             | Raft 一致性写入    | `{"Set":{"key":"a","value":"1"}}`               |
| POST | `/read`              | 本地读取          | `{"key":"a"}`                                   |
| GET  | `/membership`        | 查看当前集群成员      | -                                               |
| POST | `/change_membership` | 动态变更集群成员      | `[[2,"127.0.0.1:50052"],[3,"127.0.0.1:50053"]]` |
| GET  | `/metrics`           | Prometheus 指标 | -                                               |
